# Amazon Music Clustering Dashboard

An interactive Streamlit dashboard for clustering and visualizing Amazon Music dataset using **PCA** and **clustering techniques**.  
This project helps explore user listening patterns and visualize clusters with Plotly, Seaborn, and Matplotlib.

---

## Features
- Interactive dashboard built with **Streamlit**
- Data preprocessing using **Pandas** and **Scikit-learn**
- Dimensionality reduction with **PCA**
- Visualizations using **Plotly Express**, **Seaborn**, and **Matplotlib**
- Wide layout for better visualization
